# Image Slider – Web Project

A responsive and interactive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.  
This project demonstrates core frontend concepts such as **DOM manipulation, event handling, animations, and responsive design**.

---

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)

---

## Features
- Smooth sliding transitions
- Next and Previous navigation controls
- Auto-slide functionality (optional)
- Responsive layout for different screen sizes
- Clean and minimal UI

---

## How to Run
1. Download or clone the project repository
2. Open the `index.html` file in any modern web browser
3. Use the navigation controls to slide through images

---

## Project Structure

```
image-slider/
│
├── index.html # Slider structure
├── style.css # Styling and animations
├── script.js # Slider functionality
├── images/ # Image assets
└── README.md # Project documentation
```


---

## How It Works
- Images are arranged inside a container
- JavaScript controls the active slide using index tracking
- CSS transitions create smooth sliding effects
- Navigation buttons update the displayed image dynamically
